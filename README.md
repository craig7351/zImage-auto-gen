# Photo Album Client

這是一個基於 ComfyUI 的簡易圖片生成客戶端。

## 功能
*   連接 ComfyUI Server (自訂 IP/Port)
*   指定輸出目錄
*   使用 Z Image Turbo 工作流生成圖片

## 安裝與執行

1.  安裝依賴:
    ```bash
    pip install -r requirements.txt
    ```

2.  執行程式:
    ```bash
    python main.py
    ```

## 打包 (EXE)
本專案支援通過 GitHub Actions 自動打包。推送到 GitHub 後，請至 Actions 頁面下載 Artifact。
